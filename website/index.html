会员站，跑在自己的服务器上

OpenLayerlyPro 是开源、自托管的单画师会员站系统。发布会员作品、设置会员等级， 用收款码人工审核、Stripe 一次性付款或自动订阅收款，把高清图、PSD、笔刷包和视频 直接交付给你的粉丝——没有平台抽成，数据完全归你。

快速开始 查看源码

一台能跑 Docker 的机器即可，支持 Cloudflare Tunnel，家庭服务器无公网 IP 也能用。

terminal
# 三分钟拉起你的会员站
git clone https://github.com/39mikuu/OpenLayerlyPro.git
cd OpenLayerlyPro && cp .env.example .env
docker compose up -d
yourname.example.com

核心闭环

从发布作品到粉丝下载，一条链路走完

不需要拼接一堆 SaaS。发布、收款、审核、开通、交付，全部在一个系统里完成。

初始化站点

首次访问进入引导页，填写站点名称、创作者信息和管理员账号，配置 SMTP 与存储。

创建会员等级

配置收款码和/或 Stripe，设置会员等级与价格，发布 public / login / member 三级权限的作品。

粉丝付款

粉丝邮箱验证码登录，上传付款截图、进入 Stripe Checkout，或创建自动续费订阅。

开通与交付

人工审核或签名 webhook 自动确认后，系统事务化开通会员并发送邮件，粉丝按权限访问和下载。

功能

为「一个人的创作事业」设计

不做多画师平台、不做推荐流，只把单创作者会员站需要的东西做扎实。

三种收款方式

收款码 + 付款截图人工审核、Stripe 一次性 Checkout、Stripe 自动订阅续费，可组合使用，无卡地区也有手动续费提醒。

三级内容权限

public / login / member 三档可见性，首页限量展示，作品列表游标分页，锁住该锁的，露出该露的。

大文件交付

高清图、PSD、ZIP、笔刷包与 mp4/webm 视频附件，流式上传、Range 断点下载、内联视频播放，本地或 S3/R2 存储。

可靠的授予与反转

签名 webhook 持久化 inbox、事务化会员开通、durable outbox 邮件投递；退款与拒付自动反转对应付款期。

中 / 英 / 日三语

UI 完整 i18n，内容支持多语言与 AI 辅助翻译草稿（默认关闭、管理员手动触发），面向全球粉丝。

自托管友好

Docker Compose 一键部署，启动时自动迁移数据库；per-request nonce CSP、安全响应头与硬化备份/恢复开箱即用。

适合谁

我们把「不做什么」也写在首页

开源项目最省时间的方式，是让你在部署之前就知道它是不是你要的。

✦ 适合你，如果

  • 你是想自己掌握粉丝关系和收入的单画师 / 单创作者
  • 你想用收款码人工审核、Stripe 一次性付款或自动订阅，或三者组合
  • 你（或你的朋友）能维护 Docker Compose、PostgreSQL、SMTP 和备份
  • 你想在家庭服务器 / NAS 上跑，通过 Cloudflare Tunnel 暴露服务

当前版本不包含

  • 多画师入驻平台、内容广场或推荐流
  • 评论、点赞、收藏、关注
  • OAuth 或粉丝密码注册
  • 视频转码、HLS/DASH 与多实例高可用编排

完整的非目标清单见 README——我们宁可现在说清楚，也不想让你部署完才发现。

部署

一台小主机，就是你的全部基础设施

Docker Compose 启动：容器启动时自动执行数据库迁移，迁移失败则拒绝启动，不会带病上线。
Cloudflare Tunnel：无公网 IP 的家庭服务器、NAS、PVE 虚拟机都能对外服务。
存储随你选：本地磁盘（原子写入）或 S3 / Cloudflare R2 对象存储。
硬化备份 / 恢复：数据库与文件的备份恢复流程经过端到端验收测试。
.env
# 至少配置这两项就能跑起来
SESSION_SECRET=change-me-to-a-long-random-string
SMTP_URL=smtps://user:pass@smtp.example.com:465

# 可选：Stripe / S3 / Cloudflare Tunnel
STRIPE_SECRET_KEY=sk_live_...
APP_URL=https://yourname.example.com
