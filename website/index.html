属于画师自己的会员站，跑在自己的服务器上

发布会员作品，设置会员等级，用收款码人工审核、Stripe 一次性付款或自动订阅收款， 把高清图、PSD、笔刷包和视频直接交付给粉丝。没有平台抽成，数据完全归你。

快速开始 GitHub

一台能跑 Docker 的机器即可。支持 Cloudflare Tunnel，无公网 IP 也能部署。

$ git clone https://github.com/39mikuu/OpenLayerlyPro.git
$ cd OpenLayerlyPro && cp .env.example .env
$ docker compose up -d   # 首次访问 /admin/setup 完成初始化

工作方式

发布、收款、审核、开通、交付，在一个系统里走完，不需要拼接一堆 SaaS。

01

初始化站点

首次访问进入引导页，填写站点信息和管理员账号，配置 SMTP 与存储。

02

创建会员等级

配置收款码和/或 Stripe，设置等级与价格，发布三级权限的作品。

03

粉丝付款

粉丝邮箱验证码登录，上传付款截图、进入 Stripe Checkout，或创建自动续费订阅。

04

开通与交付

人工审核或 webhook 自动确认后，系统事务化开通会员并发送邮件，粉丝按权限访问和下载。

功能

不做多画师平台、不做推荐流，只把单创作者会员站需要的东西做扎实。

三种收款方式

收款码 + 付款截图人工审核、Stripe 一次性 Checkout、Stripe 自动订阅，可组合使用；无卡地区有手动续费提醒。

三级内容权限

public / login / member 三档可见性，首页限量展示，作品列表游标分页。

大文件交付

高清图、PSD、ZIP、笔刷包与视频附件，流式上传、Range 断点下载、内联播放，本地或 S3 / R2 存储。

可靠的授予与反转

签名 webhook 持久化、事务化会员开通、可靠邮件投递；退款与拒付自动反转对应付款期。

中 / 英 / 日三语

UI 完整 i18n，内容支持多语言与 AI 辅助翻译草稿（默认关闭，管理员手动触发）。

自托管友好

Docker Compose 部署，启动时自动迁移数据库；CSP、安全响应头与硬化备份 / 恢复开箱即用。

适合谁

在部署之前，先确认它是不是你要的。

适合你，如果

  • 你是想自己掌握粉丝关系和收入的单画师 / 单创作者
  • 你想用收款码人工审核、Stripe 付款或订阅，或三者组合
  • 你能维护 Docker Compose、PostgreSQL、SMTP 和备份
  • 你想在家庭服务器或 NAS 上跑，通过 Cloudflare Tunnel 对外服务

当前版本不包含

  • 多画师入驻平台、内容广场或推荐流
  • 评论、点赞、收藏、关注
  • OAuth 或粉丝密码注册
  • 视频转码、HLS/DASH 与多实例高可用

完整的非目标清单见 README。

部署

一台小主机，就是你的全部基础设施。

Docker Compose容器启动时自动执行数据库迁移，迁移失败则拒绝启动。
Cloudflare Tunnel无公网 IP 的家庭服务器、NAS、PVE 虚拟机都能对外服务。
存储本地磁盘（原子写入）或 S3 / Cloudflare R2 对象存储。
备份 / 恢复数据库与文件的备份恢复流程经过端到端验收测试。
# .env 至少配置这两项就能跑起来
SESSION_SECRET=change-me-to-a-long-random-string
SMTP_URL=smtps://user:pass@smtp.example.com:465
