# Site Settings

Admin path: `/admin/site`

Use site settings to configure public identity and public-site customization.

## Public Identity

- site name
- creator name
- creator bio
- creator avatar
- social links
- site logo
- site icon / favicon

The public API exposes only public site information. Custom footer code is not returned by `/api/site`.

## Custom Footer Code

Custom footer code is managed in the site settings page. It is inserted only on public pages. See [branding-and-custom-code.md](./branding-and-custom-code.md) for risks and recommended use.
